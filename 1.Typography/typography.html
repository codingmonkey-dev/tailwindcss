각 속성들을 직접 확인해보세요

📏 Font Size (폰트 크기)

text-xs (12px) - 아주 작은 텍스트

text-sm (14px) - 작은 텍스트

text-base (16px) - 기본 텍스트

text-lg (18px) - 큰 텍스트

text-xl (20px) - 더 큰 텍스트

text-2xl (24px) - 제목 크기

text-3xl (30px) - 큰 제목

text-4xl (36px) - 메인 제목

text-5xl (48px) - 거대한 제목

💪 Font Weight (폰트 굵기)

font-thin (100) - 매우 얇은 글꼴

font-extralight (200) - 아주 얇은 글꼴

font-light (300) - 얇은 글꼴

font-normal (400) - 기본 글꼴

font-medium (500) - 중간 글꼴

font-semibold (600) - 반굵은 글꼴

font-bold (700) - 굵은 글꼴

font-extrabold (800) - 매우 굵은 글꼴

font-black (900) - 가장 굵은 글꼴

📐 Line Height (줄 간격)

leading-none (1.0) - 줄 간격이 없는 텍스트입니다. 여러 줄에 걸쳐 작성하면 글자들이 서로 붙어 보입니다.

leading-tight (1.25) - 좁은 줄 간격의 텍스트입니다. 컴팩트한 느낌을 주고 싶을 때 사용합니다.

leading-normal (1.5) - 기본 줄 간격의 텍스트입니다. 가장 일반적으로 사용되는 줄 간격입니다.

leading-relaxed (1.625) - 여유로운 줄 간격의 텍스트입니다. 읽기 편하고 시원한 느낌을 줍니다.

leading-loose (2.0) - 느슨한 줄 간격의 텍스트입니다. 매우 시원하고 여유로운 느낌을 줍니다.

🔤 Letter Spacing (글자 간격)

tracking-tighter - 매우 좁은 글자 간격

tracking-tight - 좁은 글자 간격

tracking-normal - 기본 글자 간격

tracking-wide - 넓은 글자 간격

tracking-wider - 더 넓은 글자 간격

tracking-widest - 가장 넓은 글자 간격

🎨 Text Decoration (텍스트 장식)

underline - 밑줄 텍스트

overline - 윗줄 텍스트

line-through - 취소선 텍스트

no-underline - 밑줄 제거 (링크 스타일)

📍 Text Alignment (텍스트 정렬)

text-left - 왼쪽 정렬

text-center - 가운데 정렬

text-right - 오른쪽 정렬

text-justify - 양쪽 정렬 - 이 텍스트는 양쪽 정렬을 보여주기 위해 길게 작성되었습니다. 문단의 양쪽 끝이 정렬되어 보입니다.

🔄 Text Transform (텍스트 변환)

uppercase - 모두 대문자로 변환

LOWERCASE - 모두 소문자로 변환

capitalize - 첫 글자만 대문자로

normal-case - 기본 케이스

📱 반응형 타이포그래피

반응형 텍스트 - 화면 크기에 따라 글자 크기가 변합니다

반응형 굵기 - 화면 크기에 따라 글자 굵기가 변합니다

반응형 줄 간격 - 화면 크기에 따라 줄 간격이 변합니다. 이 텍스트는 여러 줄에 걸쳐 작성되어 줄 간격의 차이를 확인할 수 있습니다.
